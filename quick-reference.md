## ⌨️ Markdown Cheatsheet

**Center Everything:**

```html
<div align="center">... content ...</div>
```

**Responsive Images:**

```html
<img src="image.png" width="100%" />
```

**Custom Link Button:**

```html
<a href="URL">
  <img
    src="https://img.shields.io/badge/LABEL-COLOR?style=for-the-badge&logo=ICON"
  />
</a>
```

## ⚠️ Do's and Don'ts

- ✅ **DO** use `width` attributes to prevent layout shift.
- ✅ **DO** use `?raw=true` for local images.
- ❌ **DON'T** use JavaScript (it won't run).
- ❌ **DON'T** use iframes (blocked by GitHub).
