## Reporting a Vulnerability

We take the security of our software seriously. If you have found a security vulnerability in this project, or any related project under the **Ashraf Morningstar** organization, please report it to us as detailed below.

### How to Report

Please do **not** report security vulnerabilities through public GitHub issues.

Instead, please report them via email to **contact@ashrafmorningstar.com** (replace with your actual email) or use the [GitHub Private Vulnerability Reporting](https://github.com/AshrafMorningstar/AshrafMorningstar/security/advisories/new) feature if enabled.

You should include:

- A description of the location and potential impact of the vulnerability.
- A detailed description of the steps required to reproduce the vulnerability (POC scripts, screenshots, and compressed packet captures are all helpful to us).

### Response Timeline

- We will acknowledge receipt of your report within 24 hours.
- We will provide you with a progress update every 5 working days.
- We credit fellow security researchers for their contributions (unless you prefer to remain anonymous).

Thank you for helping keep the community safe!
